# app-games
A simple Crud Angular and mysql

# Quick Start 

**Install dependencies for server**

```
npm install
npm run build
```

**Run the client & server with concurrently**
```
npm run dev
```
**Run the Angular client only**
```
ng serve
```
#Server runs  on http://localhost:3000 and client on http://localhost: 4200
